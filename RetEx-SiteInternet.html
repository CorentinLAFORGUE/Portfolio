Site internet

Rapport économique Delpeyrat porté sur un site internet

Mes expériences

Compétences développées

  • Analyse de besoin
  • Programmation web

L'objectif de ce projet était de réaliser un site internet dans son intégralité. Ce site devait présenter le rapport économique portant sur Delpeyrat (Voir "Analyse économique"). Le site devait comporter 1 page d'accueil en anglais, ainsi qu'une page par partie abordée par le rapport, et une page de mentions légales avec un formulaire de contact.

Cahier des charges

  • Recueil de besoins
  • Programmation web
  • Site "responsive" (affichage s'adapte à la taille de la page du navigateur)

Technologies et Méthodes utilisées

  • Langage HTML
  • Langage CSS
